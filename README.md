# UAC Care System Load Analytics

Interactive **Streamlit** dashboard for monitoring the U.S. Department of Health and Human Services (HHS) **Unaccompanied Alien Children (UAC) Program** — tracking system capacity, intake-to-discharge flow, backlog accumulation, and surge-risk KPIs from raw daily operational data (2023–2025).

## Overview

The UAC Program is a federally mandated care pipeline: children apprehended by CBP are transferred to HHS for medical screening, sheltering, and eventual placement with vetted sponsors. This project turns raw daily operational counts into a structured analytics framework that lets stakeholders **monitor system load, detect capacity strain early, and evaluate whether inflow and outflow are in balance** — instead of relying on reactive, headline-count monitoring.

**Key result:** analysis of 1,075 days of data (Jan 2023 – Dec 2025) shows the system-wide care load declined ~55% over the period, driven by a ~59% drop in new intake — not by any breakdown in HHS processing capacity. Full findings are in [`UAC_Research_Report.docx`](./UAC_Research_Report.docx).

## Dashboard Modules

| Module | What it shows |
|---|---|
| **KPI Summary Cards** | Total Children Under Care, Net Intake Pressure, Care Load Volatility Index, Backlog Accumulation Rate, Discharge Offset Ratio |
| **System Load Overview Pane** | Total system load trend, 7/14-day rolling averages, sustained high-load periods, early-vs-late period comparison |
| **CBP vs HHS Load Comparison** | Side-by-side trend of CBP custody load vs. HHS care load |
| **Net Intake & Backlog Trends** | Daily net intake (transfers − discharges) and accumulated backlog over time |

**User controls:** date range selector, metric show/hide toggles, and time granularity filter (Daily / Weekly / Monthly).

## Dataset

Source: `HHS_Unaccompanied_Alien_Children_Program.csv` — daily operational counts reported by HHS.

| Column | Description |
|---|---|
| Date | Reporting date |
| Children apprehended and placed in CBP custody | Daily intake volume |
| Children in CBP custody | Active CBP care load |
| Children transferred out of CBP custody | Flow into HHS system |
| Children in HHS Care | Active HHS care load |
| Children discharged from HHS Care | Successful sponsor placements |

## Methodology

1. **Data Ingestion & Structuring** — load raw CSV, clean numeric fields, convert dates, build a complete daily calendar index (raw data only has report-days, not every calendar day).
2. **Data Quality & Validation** — flag missing/duplicated dates, validate logical constraints (Transfers ≤ CBP custody, Discharges ≤ HHS care), flag reporting anomalies for transparency (kept, not silently dropped).
3. **Derived Capacity Metrics** — Total System Load (CBP + HHS), Net Daily Intake (Transfers − Discharges), Care Load Growth Rate (day-over-day % change), Backlog Accumulation (floored running total of net intake).
4. **Trend & Temporal Analysis** — daily/weekly/monthly trends, sustained high-load period detection, early-vs-late timeline comparison.
5. **Pressure & Stress Identification** — 7-day and 14-day rolling statistics, a normalized Care Load Volatility Index (coefficient of variation), and prolonged strain window detection.

## KPIs

| KPI | Description |
|---|---|
| Total Children Under Care | System-wide responsibility (CBP + HHS) |
| Net Intake Pressure | Inflow vs. outflow imbalance |
| Care Load Volatility Index | Stability of the system (std/mean of HHS care load) |
| Backlog Accumulation Rate | Sustained care pressure (daily change in backlog) |
| Discharge Offset Ratio | Ability to relieve load (discharges ÷ transfers) |

## Key Findings

- System-wide care load fell from an average of **8,364** (early period) to **3,766** (late period) — a **55% decline**.
- The decline is **intake-driven**: daily intake, CBP custody, transfers, HHS care, and discharges all fell by a similar 55–73% in parallel — ruling out a processing bottleneck as the cause.
- 2025 average daily intake (12.9) is roughly **9% of the 2023 level** (115.0) — an ~89% collapse in new apprehensions over two years.
- The system is currently stable (Care Load Volatility Index ≈ 0.0025) with a shrinking backlog (Backlog Accumulation Rate ≈ −2.4/day) and a near-balanced Discharge Offset Ratio (≈ 0.95).
- Despite current low load, historical data shows the system can swing 2–3x within 18 months — supporting the case for maintained surge-readiness rather than permanent capacity reduction.

Full analysis, methodology notes, and stakeholder recommendations: [`UAC_Research_Report.docx`](./UAC_Research_Report.docx).
